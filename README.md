#### AI-Powered Personalized Learning Platform###### An AI-based web application that detects topic-wise knowledge gaps in students and provides personalized learning recommendations.##### Features* Topic-level performance analysis* Knowledge gap detection* Personalized learning recommendations* Progress tracking dashboard* Simple web interface (Flask-based)##### Tech Stack* Backend: Python, Flask, Scikit-learn* Frontend: HTML, CSS, JavaScript* Data Handling: Pandas, NumPy* Database: SQLite##### How It Works* Student assessment data is uploaded* System analyzes topic-wise scores* Weak concepts are detected* Recommendations are generated* Progress is visualized on dashboard
